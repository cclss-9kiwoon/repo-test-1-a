# 인디 게임 2 — 게이미피케이션 고민 카드

## 핵심 질문

MVP 검증 후 게임성을 어떻게 더 깊게 만들 것인가?

---

## 우선순위 높음 (다음 스프린트)

### 1. 세이브/로드 시스템
- localStorage 기반 자동 저장
- 골드, 업그레이드 레벨, 현재 스테이지 보존
- **기대 효과**: 플레이어 이탈 방지, 재방문 유도
- **난이도**: 낮음

### 2. 오프라인 보상
- 접속하지 않은 시간만큼 골드 자동 적립
- 최대 8시간분 골드까지
- **기대 효과**: 매일 접속하는 습관 형성 (idle 게임의 핵심)
- **난이도**: 낮음

### 3. 보스 몬스터
- 5스테이지마다 보스 등장 (스테이지 5, 10, 15, 20)
- 일반 적보다 HP/ATK 3배, 특수 속성 랜덤
- 보스 처치 시 특별 보상 (골드 5배 + 업그레이드 재료)
- **기대 효과**: 중간 목표 제공, 성취감 극대화
- **난이도**: 중간

---

## 우선순위 중간 (검증 후)

### 4. 진화 시스템
- 특정 스테이지 클리어 시 캐릭터 진화
- 소 → 한우 → 와규 (HP 대폭 증가)
- 돼지 → 흑돼지 → 이베리코 (ATK 대폭 증가)
- 닭 → 토종닭 → 불사조 (속도 + 특수 능력)
- **기대 효과**: 장기 목표, 수집 욕구
- **고민**: 진화 조건을 스테이지로 할지, 재료 수집으로 할지

### 5. 스킬 시스템
- 각 캐릭터에 액티브 스킬 1개씩
- 소: 도발 (적 공격을 3턴 자신에게 집중)
- 돼지: 돌진 (2배 데미지 1회)
- 닭: 속사 (3연속 공격)
- 쿨타임제, 수동 발동
- **기대 효과**: 전략적 개입 포인트, idle의 단조로움 해소
- **고민**: idle 게임에서 수동 조작이 많으면 피로감?

### 6. 장비/파츠 시스템
- 적 처치 시 랜덤 드롭
- 무기(ATK+), 방어구(HP+), 장신구(특수 효과)
- 등급: 일반 → 고급 → 희귀 → 전설
- **기대 효과**: 랜덤성으로 인한 도파민, 파밍 동기
- **고민**: 인벤토리 UI 복잡도

---

## 우선순위 낮음 (장기 로드맵)

### 7. PvP / 랭킹
- 다른 플레이어 팀과 자동 대전
- 주간 랭킹, 보상 지급
- **기대 효과**: 경쟁 요소, 소셜 리텐션
- **고민**: 서버 필요, MVP 범위 초과

### 8. 스킨 시스템
- 캐릭터 외형 변경 (순수 코스메틱)
- 특별 이벤트나 업적 달성 시 획득
- **기대 효과**: 수집 욕구, 수익화 가능성
- **고민**: 아트 리소스 필요

### 9. 이벤트 던전
- 기간 한정 특별 스테이지
- 특수 보상 (한정 장비, 스킨)
- **기대 효과**: 라이브 서비스 느낌, FOMO 활용
- **고민**: 콘텐츠 제작 비용

## 게임성 핵심 고민

### 자동 vs 수동의 균형
- idle 게임의 매력은 "놔두면 알아서 진행"
- 하지만 완전 자동이면 게임이 아니라 시뮬레이션
- **현재 해결책**: 속성 변경만 수동 → 충분한가?
- **추가 고려**: 스킬 시스템으로 "가끔 터치" 유도

### 업그레이드 만족감
- 숫자가 올라가는 것만으로 만족스러운가?
- 시각적 피드백 필요 (이펙트, 흔들림, 색상 변화)
- 마일스톤 축하 연출 (스테이지 10 클리어 시 특별 화면)

### 속성 전략의 깊이
- 현재: 3속성 상성 → 단순하지만 명확
- 확장 가능: 속성 4~5개 추가 (빛, 어둠)? 아니면 3개로 유지?
- 캐릭터별 고유 속성 vs 팀 공통 속성 → 현재는 팀 공통

### 과금 모델 (Unity 전환 시)
- 광고 보상 (2배 골드, 부활 등)
- 배틀 패스 (시즌별 보상 경로)
- 스킨 구매
- **주의**: P2W는 피해야 함 → 코스메틱 위주
